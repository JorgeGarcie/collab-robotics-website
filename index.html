Project Goal

Integrate a state machine for logic, process audio to detect commands, recognize visual inputs, perform pose estimation, and plan navigation for object retrieval.

Tasks

Task 1: Retrieve Fruit and Go Back

The robot navigates to the object, picks it up, and returns to its starting position.

Methods

  • State Machine: Manages eight logic states.
  • Language: Records and processes audio to determine commands.
  • Perception: Recognizes visual inputs for object identification.
  • Pose Estimation: Transforms and commands poses.
  • Navigation: Plans motion and retrieves objects using path planning.
