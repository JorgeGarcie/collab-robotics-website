Project Goal

Our project aimed to program a Locobot to engage in different fruit-gathering tasks utilizing perception, obstacle avoidance, and navigation.

Key Features of the Interbotix LoCoBot WX250:

  • Create 3 Mobile Platform: A robust and flexible base for mobile robotics.
  • RealSense Depth Camera: Enables 3D perception for precise environment mapping and object detection.
  • 360-degree 2D LIDAR: Provides comprehensive spatial awareness for accurate navigation.
  • WidowX 250 6DOF Robot Arm: Delivers high-precision manipulation, perfect for tasks that require dexterity.

Tasks Overview:

  • Task 1: Retrieve fruit and return to the starting point.
  • Task 2: Pick fruit and place it in a basket.
  • Task 3: Detect human pointing and pick up the indicated fruit.
